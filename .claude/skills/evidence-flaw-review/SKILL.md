# 證據瑕疵審查（skill）

盧大律師武器③（攻證據取得瑕疵）的操作技能，由 EVD 證據法律師主導。對支持證據表中的**每一項
證據**跑一遍下列 checklist，產出可攻擊點。核心問句：**「這個證據能不能用？能不能單獨用？」**

## Checklist（逐項打勾）

### 1. 證據定性
- 供述證據（自白／共同被告陳述／證人證言）？還是物證／書證／鑑定／數位證據？
- 供述證據的瑕疵空間最大，優先深查。

### 2. 傳聞法則（刑訴 §159、§159-1~5）
- [ ] 是「被告以外之人於審判外之陳述」嗎？→ 原則無證據能力。
- [ ] 落入哪個例外？（§159-1 向法官/檢察官、§159-5 同意…）
- [ ] 警詢陳述原則屬傳聞；偵訊陳述須具結且無顯不可信情況。

### 3. 補強法則（刑訴 §156-II ＋ 釋字第582號）
- [ ] 這是**自白**或**共同被告不利陳述**嗎？→ 不得為有罪之**唯一**證據。
- [ ] 釋字582：共同被告陳述＝證人證言，須**具結**＋**對質詰問**；未具結則能力/證明力受限。
- [ ] 補強證據存在嗎？**補強的射程夠到「犯罪事實（尤其主觀構成要件）」嗎？**
  → 只補強「客觀環境可疑」不夠；客觀可疑 ≠ 被告主觀明知。

### 4. 自白任意性（刑訴 §156-I、§98）
- [ ] 取得過程有無強暴、脅迫、利誘、詐欺、疲勞訊問、違法羈押？→ 非任意性自白無證據能力。

### 5. 違法取證排除（刑訴 §158-4 權衡）
- [ ] 搜索、扣押、通訊監察有無令狀、有無逾越範圍？
- [ ] 違背法定程序時，依 §158-4 權衡人權保障與公共利益 → 證據能力如何。

### 6. 對造/犯罪集團單方記載（證明力層次）
- [ ] 是犯罪集團內部帳冊/表格對第三人的記載嗎？（會誇大/對己有利）
- [ ] 不得以對造單方記載**回推被告的故意或從屬關係**。

### 7. 數位證據的同一性/完整性
- [ ] 手機/後台資料的取得、保管鏈、鑑識程序是否完整？暱稱人別是否確認（如「Jay Lai」）？

## 產出格式（填入支持證據表的「可攻擊點」欄）

對每項證據輸出：
1. **證據能力結論**：有／無／有瑕疵（依 §159/§156/§158-4）。
2. **證明力結論**：能否單獨採認、補強射程是否及於主觀、有無單方記載問題。
3. **一句話攻擊點**：交盧大放進三層辯證的武器③。

## 本案高價值標的
- **D1（周正桓 27-3）**：共同被告、**拒絕具結**、無錄音/第三人/書面 → 釋字582＋§156-II，
  不得單獨定罪；補強僅及客觀環境。
- **水房王藝縈電腦表格**：對造單方記載，不得回推賴宗杰故意（第四輪教訓）。
- **黑名單一覽表 / 後台網域分析**：物證/數位證據，攻擊點在「證明客觀事實 ≠ 證明主觀明知」。
