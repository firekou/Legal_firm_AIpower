# 盧氏三層辯證分析法

> 盧大律師的核心方法論。與 `對抗式多輪會議無罪推論法.md` 互補：
> **三層辯證法 = 對「單一事實現象」的靜態結構解剖工具；多輪會議法 = 動態迭代流程。**
> 每一輪會議中要深掘某個特徵點時，就用三層辯證法把它拆開。

---

## 一、核心哲學：事實不是被發現的，是被「呈現」的

> 一個絕對的事實縱然存在，仍需要透過「證據」與「辯證」來支持它的**呈現**。

盧大律師的世界觀有三個層次，必須分清楚：

| 層次 | 名稱 | 說明 |
|------|------|------|
| 1 | **物理絕對真實** | 當時實際發生的事。無人能完整重現，法庭上**不可得**。 |
| 2 | **事實現象** | 經由證據與辯證「呈現」出來的事實。這是法庭真正在處理的東西。 |
| 3 | **辯證與證據** | 支撐「事實現象」站立的地基。 |

**關鍵洞察：**
- 我們在法庭上攻防的，永遠是**第 2 層（事實現象）**，不是第 1 層。
- 事實現象與物理絕對真實之間**本來就存在差距**——因為它是被「呈現」出來的，不是被完整
  重現的。
- **不同角度的解釋，會改變支撐事實現象的辯證。** 同一批證據，換一個解釋框架，可以撐起
  一個完全不同的事實現象。

**打地基比喻：** 一個事實現象像一棟建築，靠底下的辯證與證據當地基撐著。我們的工作不是
去爭論「這棟樓該不該存在」，而是去檢查**地基裡哪幾根樁是承重的**——抽掉它，樓就塌，
事實現象就得重新蓋成另一個樣子。

**這與誠實鐵律不衝突：** 我們不是捏造事實，而是主張——檢方呈現的「事實現象」，其地基
（辯證與證據）不足以支撐它站到「無合理懷疑」的高度；同一批證據，用對被告有利且同樣合理
的解釋框架，會呈現出另一個事實現象。

## 二、三層架構（三張表）

針對任何一個要處理的**事實現象**（例如「賴宗杰明知客戶從事賭博」），建立三張表：

### 表一：辯證表
支撐這個事實現象的**所有辯證（命題／主張）**逐一列出。

| 欄位 | 說明 |
|------|------|
| 辯證編號 | A1, A2, A3… |
| 命題 | 一句話寫出這個辯證主張什麼（支撐事實現象的一塊磚） |
| 對事實現象的作用 | 這塊磚撐住的是哪一部分 |
| **承重度** | 🔴承重（抽掉則事實現象崩）／🟡輔助／🟢裝飾（可有可無） |

### 表二：辯證立場表
整理**各方對每一個辯證的觀點**。同一個辯證，不同角色看法不同——這是找施力點的地圖。

| 欄位 | 說明 |
|------|------|
| 辯證編號 | 對應表一 |
| 檢方立場（PRO） | 檢方如何主張這個辯證成立、代表什麼 |
| 辯方立場（DEF） | 辯方如何質疑或重新解釋 |
| 技術/中立觀點（INV/JDG） | 客觀技術事實、法官傾向 |
| 當前傾向 | 有利／中性/待補強／不利 |

### 表三：支持證據表
整理**每一個辯證所依附的證據**，特別關注**證據的取得過程**（這是最常見的施力點）。

| 欄位 | 說明 |
|------|------|
| 辯證編號 | 對應表一 |
| 證據來源 | 哪份筆錄/鑑識報告/通訊紀錄/帳冊（標檔名頁碼） |
| 證據種類 | 供述證據（自白/證言）／物證／書證／鑑定／數位證據 |
| **取得過程 / 證據能力** | 如何取得？有無具結？有無傳聞問題？有無違法取證？ |
| 可攻擊點 | 這個證據在能力或取得上的瑕疵 |

## 三、操作方法：找施力點、拆積木、重構、導向

### 步驟 1：全面審視，建三表
把目標事實現象的辯證、立場、證據**窮舉**進三張表。沒有全面審視，就找不到真正的承重樁。

### 步驟 2：標記施力點（承重積木）
在表一標出 🔴承重辯證，在表三標出可攻擊的證據。**施力點 = 承重且可攻擊的交集。**

### 步驟 3：選擇施力方式（三種武器）

| 武器 | 打哪張表 | 作法 | 效果 |
|------|---------|------|------|
| **① 強調有利立場** | 表二 | 在辯證立場中，凸顯對被告有利、且同樣合理的解釋框架 | 讓辯證的意義倒向我方 |
| **② 攻辯證瑕疵** | 表一 | 指出辯證本身的邏輯跳躍或**定義錯置**（如「博奕」≠「賭博」） | 辯證命題本身不成立 |
| **③ 攻證據取得瑕疵** | 表三 | 證明證據的**取得過程有瑕疵**（未具結、傳聞、違法取證、單方記載）→ 證據能力/證明力不足 | 抽掉辯證的地基，辯證隨之倒 |

> **武器 ③ 是盧氏方法最鋒利的一把。** 因為它不跟你爭「事情是不是真的」，而是主張「這個
> 用來證明它的證據，依證據法則不能用（或不能單獨用）」——一旦證據被壓制，它撐著的辯證
> 就失去地基，連帶那部分事實現象就得重蓋。

### 步驟 4：積木移除 → 事實現象重構
抽掉承重辯證後，原本的事實現象**無法再以原貌站立**，局面重構為一個新的事實現象——通常是
對被告有利的那一個。

### 步驟 5：導向目標結果
你要的不一定是「全部推翻」，而是**導向一個特定結果**。先決定終點，再回推「要抽哪幾塊積木」。

**導向矩陣：**

| 想要的轉變 | 主攻的施力點 | 典型手法 |
|-----------|------------|---------|
| 有罪 → 無罪 | 抽掉**主觀構成要件**（故意/明知）的承重辯證 | 武器②定義錯置＋武器③壓制唯一明知證據 |
| 重罪 → 輕罪 | 抽掉**加重要件/共謀連結**的承重辯證 | 武器①責任分層＋武器③壓制共謀證據 |
| 有罪 → 緩刑/易科 | 保留成罪但抽掉**惡性/情節加重** | 武器①凸顯有利量刑因子（自白、無不法意圖、犯罪支配薄弱） |
| 共同正犯 → 幫助犯 | 抽掉**犯罪支配/功能性貢獻**的辯證 | 武器①②降低參與程度 |

## 四、與「對抗式多輪會議無罪推論法」的整合

兩者不是兩套方法，是**結構工具 × 流程引擎**：

```
多輪會議法（流程）
 每一輪引入新證據 → PRO/DEF 攻防 → JDG 定錨
 │
 └─ 當某個特徵點成為焦點（D 點/槓桿點）時，
 對它啟動 ▼
 盧氏三層辯證法（結構工具）
 建三表 → 標施力點 → 三武器施力 → 事實重構 → 導向
```

對照關係（舊方法的用語 → 新方法的位置）：

| 對抗式多輪會議法 | 盧氏三層辯證法 |
|-----------------|--------------|
| D 點 / 槓桿點 | 承重辯證（表一 🔴） |
| 錨點清單 A1, A2… | 辯證表 + 支持證據表 |
| 破防三武器（定義消除／證據法則壓制／事實校準） | 三種施力（②攻辯證／③攻證據／①強調立場＋事實校準） |
| 結論變化盤點 | 積木移除後的事實重構 |

## 五、Worked Example：用三層辯證法拆解 D1

**目標事實現象：**「賴宗杰主觀上明知客戶從事賭博/洗錢犯罪」（檢方最想立起來的那棟樓）

### 表一：辯證表

| 編號 | 命題 | 承重度 |
|------|------|--------|
| A1 | 周正桓曾告知賴宗杰客戶涉博奕，賴答「業界都這樣」（D1） | 🔴 **唯一直接明知證據** |
| A2 | 涉娛樂城手續費佔 81.67% | 🟡 輔助（客觀環境） |
| A3 | 對話中出現通博/HOYA 等娛樂城名稱 | 🟡 輔助 |
| A4 | 30 年業務資歷，應能辨別 | 🟢 經驗法則推論 |

→ **施力點鎖定 A1。** A2/A3/A4 都只能證明「客觀環境可疑（可知）」，唯有 A1 直指「已知」；
抽掉 A1，這棟「明知」的樓就失去唯一承重樁。

### 表二：辯證立場表（A1）

| 檢方（PRO） | 辯方（DEF） | 中立（JDG） |
|------------|------------|------------|
| 內部工程師親證，不依賴水房共謀，直證明知 | 「博奕」是法律外行的概括印象，非「賭博犯罪」之認識；「業界都這樣」可指客訴/函調常態 | 若採信且經補強，足認概括認識；但受補強法則限制 |

### 表三：支持證據表（A1）

| 證據來源 | 種類 | 取得過程 / 證據能力 | 可攻擊點 |
|---------|------|-------------------|---------|
| 周正桓 27-3 偵訊陳述 | 供述（共同被告） | **拒絕具結**（「不想傷害賴宗杰」）；無錄音、無第三人、無書面 | 未具結共同被告陳述→釋字582＋刑訴§156-II 補強法則，不得為唯一證據；補強證據（A2/A3）僅及客觀環境，不及主觀明知 |

### 施力與重構（五重削弱＝抽 A1 這根樁）

1. 武器③：周正桓**未具結**→補強法則，A1 不能單獨定罪。
2. 武器②：**「博奕」≠「賭博」**定義錯置，A1 命題內容不含賭博犯罪之認識。
3. 武器②：周正桓自承「對法律不夠瞭解」→傳遞者無正確認知，A1 傳遞的不是明知。
4. 武器①：史育靜證實「娛樂城＝遊戲平台」是產業鏈通用語→A1 是全鏈盲點，非個人明知。
5. 武器①：客觀紅旗（帳戶備註、後台串接）不在賴宗杰視野→無容任基礎。

**重構後的事實現象：** 「金恆通內部對客戶有概括的、法律外行的博奕懷疑」——**但這撐不起
「賴宗杰主觀明知賭博/洗錢犯罪」那棟樓**。事實現象由「明知」重構為「認知盲點下的合法代收
認知」。

**導向：** 主觀構成要件的承重樁被抽掉 → 洗錢/賭博幫助犯的故意不足 → 導向重罪無罪。

## 六、誠實鐵律（貫穿全程，與既有方法一致）

1. **不為達標而灌水機率**——地基不穩的重構，會在法庭上塌。
2. **分罪名估計**——每個罪名是一棟樓，分開拆、分開估。
3. **每根支柱都要站得住卷證**——施力點必須真的可攻擊，不能靠喊。
4. **硬點正面承擔**——當事人自身、有書面的行為（黑名單提議）是自己蓋的承重樁，抽不掉，
   只能改導向（成罪但止損為輕罪/緩刑），不可假裝拆除。
