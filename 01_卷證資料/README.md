## 命名規則（若日後擷取全文入庫）

```
筆錄/{編號}_{姓名}/{原檔名去括號}.md
例：筆錄/26_賴宗杰/26-3_111.06.30_偵訊筆錄.md
```

每份檔頭請標註：來源 Drive ID、卷別頁碼、擷取日期。

## 為什麼卷證原檔不入庫

- 掃描 PDF 體積大、為二進位，不適合 git 版本控管。
- Google Drive 為卷證的單一真實來源（single source of truth）。
- 卷證含個資與偵查資料，以連結引用、需要時再取，較利於權限控管。
