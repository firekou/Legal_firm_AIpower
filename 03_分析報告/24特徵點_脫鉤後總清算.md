# 24 特徵點 — 事實修正後的辯方總清算

**適用的三項事實修正：**
1. A3 假：汪晉緯**不認識**賴宗杰（無第三方橋接蔡居安↔賴宗杰）
2. A5 假：晉珈／水房**不能指揮**金恆通；「和解部門」是水房內部機制
3. 賴宗杰 ⊥ 王藝縈：**完全無互動、互不認識**

**總體效果：** 「金恆通＝JIAPAY 水房組織共謀」結構瓦解，全案降格為「對等客戶—服務商」
關係。檢方失去「水房共謀→必然知情」的反推捷徑，主觀明知須以賴宗杰**自身**事證獨立證明。

**「定義消除機會」分級：**
- 🟢 **可消除**＝有強力定義空間，能將不利事實重新詮釋為中性／合法
- 🟡 **可弱化**＝能減輕但無法完全消除，影響的是罪責輕重
- 🔴 **難撼動**＝賴宗杰自身行為，定義空間有限，是辯護需正面承擔的硬點

## 總結：脫鉤後的無罪路徑圖

### 撼動原始事實的三道定義槓桿

```
槓桿一【結構定義】：金恆通⊥水房（賴⊥王、A3/A5 倒塌）
 → 將全案從「組織犯罪共謀」重新定義為「對等客戶—服務商關係」
 → 消除 F-21（崩塌）、弱化 F-08/F-20、去除所有「水房共謀」加重

槓桿二【主觀定義】：遊戲點數 vs 現金板外觀難辨（F-09 樞紐）
 → 將「明知賭博」重新定義為「認知盲點下的合法代收認知」
 → 若成立，連帶消解 F-10、F-11、F-15、F-23，全案降為過失
 → 這是無罪與否的總開關

槓桿三【分工定義】：黑名單操作、函調字號、商家接洽的執行者
 → 將賴宗杰重新定義為「業務招攬者」，把設計／操作責任
 歸於周正桓（系統）與賴永三（帳戶決策）
 → 弱化 F-18、F-19，切割 F-01/F-03
```

### 辯方須正面承擔、定義空間最小的三個硬點

| 硬點 | 為何難消除 | 殘餘辯護 |
|------|-----------|---------|
| **F-17 黑名單提議** | 賴宗杰自承提議，行為無法否認 | 只能守「目的是防三方詐騙」，把濫用歸執行層 |
| **F-09 認知盲點（雙面）** | 既是最大機會也是最大風險——若檢方以自身事證證明明知，全案崩 | 主攻外觀難辨＋無水房反推，但須逐一拆解通博/HOYA/球板 |
| **F-18/19 黑名單細節** | 含被害人資料＋函調字號，個資違法客觀存在 | 責任轉嫁周正桓，但金恆通整體個資風險難全脫 |

### 結論

脫鉤後，**24 點中：**
- **3 點**因脫鉤直接弱化或崩塌（F-21 崩、F-08/F-20 弱化）
- **5 點**屬獨立硬點需正面承擔，但其中 4 點（F-07/F-10/F-18/F-19）仍有 🟡 定義弱化空間
- **僅 F-17 與 F-09 一面**是定義空間最小的真正硬點
- **F-09 是總開關**：守住「認知盲點」則全案降為「服務商審查過失」，賴宗杰有相當無罪空間；
  守不住則 F-09 反向成為主觀明知的支柱

**最有利的事實是：** 失去水房共謀的反推捷徑後，檢方要獨立證明賴宗杰「明知客戶賭博詐欺」
的舉證難度大幅升高。
