# 第二輪：交叉詰問（INV 金流數據介入）

> 本輪 INV 全面投入 myhomepei.com 與 60.75 億金流數據，PRO 與 DEF 據此重新攻防，多點立場移動。

### 重大轉折一：myhomepei.com 同一網域（衝擊 F-08、F-03、F-09）

**INV：** N 開頭 6 家公司接收交易網址**全部是 myhomepei.com**，而賴宗杰在 26-4 自承
「N 開頭公司在同一個群組內討論」。技術事實：它們不是各自獨立的公司，而是同一個後台、
同一個聯絡群組。

**PRO：** 這擊破 DEF 在 F-08 的「各家獨立、多由周正桓接洽」辯護。6 家共 296,741 筆、
15.2 億，他是審核者，不可能不知這是同一組人。

**DEF（立場調整）：** 退守——承認賴宗杰知道 N 開頭是「同一系統商管理的多家店」，但這
正是他認知的「系統商代管多店家」的正常商業模式（26-4：「系統商不會跟我們簽約，只是
幫店家管理系統」）。知道是同一系統商≠知道都是賭博。

**JDG：** F-08 從「中性」**移向不利被告**——同一網域+同一群組+本人審核，「不知是同一組
人」的辯護不成立；但「是否知道為賭博」仍留有空間。

### 重大轉折二：黑名單與被害人（衝擊 F-17、F-18、F-19）

**INV：** 黑名單一覽表掛在金恆通自有網域（pay.ufo.com.tw，IP 210.65.47.7，金恆通註冊），
內含被害人姓名、身分證、銀行帳號、檢警函調字號。黑名單 API 與代收介面串接，消費者輸入
身分證+帳號比對後才放行匯款。

**PRO：** 黑名單實際功能是「過濾掉已報警的被害人」，保護賭博/詐欺平台。賴宗杰是提議者
（F-17），且親自轉傳含「多為報警圈存消費者」的規範文件（F-19）。三點應合併認定為積極協助。

**DEF：** 「防三方詐騙」與「過濾被害人」在技術上是同一個動作（比對身分證擋掉特定人），
無法從功能反推目的。關鍵仍是賴宗杰主觀上認為被擋的是「詐騙慣犯」還是「被害人」。

**JET：** 我設黑名單時想的是有人專門假裝消費者來騙平台，不是要擋真正被害人。

**JDG：**
- F-17 維持「中性偏不利」（提議事實無爭議，目的有合理懷疑空間）。
- F-18 從「中性偏不利」**移向不利**——函調字號＝來自警方＝對象是報案被害人。
- F-19 從「中性」**移向不利**——「多為報警圈存消費者」白紙黑字，轉傳即難諉為不知。

### 重大轉折三：F-20「那個人安」身分（最關鍵攻防）

**PRO：** 指認紀錄表中賴宗杰指認「編號 2 是陳先生（那個人安）」，但編號 2 本名是**蔡居安**。
且賴宗杰將欣利(NB)、怡誠(NC)、明洋(NF)遭警調的訊息傳給他——這三家正是 myhomepei.com
同網域客戶。

**DEF：** (1) 賴宗杰主觀上一直認為對方是「做菸酒的陳先生」；(2) 若真共謀，不會在偵訊中
說「忘記了」這種對自己不利的話；(3) 傳送的是「這些公司被警調」的事實通知，性質接近
業務告知。

**INV：** 待查——需蔡居安（1-0~1-5）與王藝縈筆錄交叉比對，確認蔡居安是否為 N 開頭公司
實際聯絡人。目前金流表只能證明 N 開頭同網域。

**JDG：** F-20 從「不利被告」**微調為中性偏不利**——核心繫於「那個人安＝蔡居安且賴宗杰
知情」的證明。**這是本輪唯一對被告略微好轉的點。**

### 其他點的第二輪移動

- **F-07：** INV 確認「零 STR 申報」後，JDG 從「中性」**移向中性偏不利**。
- **F-23：** INV 確認後台分析是 111.06.29 查扣後製作，賴宗杰無後台 IP 軌跡。JDG **維持有利被告**。
- **F-09：** PRO 以 myhomepei 串接 20+ 娛樂城反推盲點不可信。DEF 守住「業務端看不到後台串接」。JDG **維持中性**。

# 第三輪：法官主導收斂

JDG 總結原則重申：**無罪推定、罪疑唯輕、故意需「認識且容任」而非「應知未知」**。據此三分：

### A. 對被告有利（合理懷疑足夠 / 證明力低）— 6 點
F-04（稱謂）、F-05（自稱經理，反證掛名）、F-12（通訊工具中性）、F-16（球板私人行為）、
F-22（私人認識）、F-23（事後數據不回推主觀）

### B. 中性／待補強（繫於尚未調閱之證據）— 8 點
F-01（掛名）、F-02、F-06、F-09（認知盲點，核心主觀）、F-11、F-13（人別待證）、F-14、F-21

### C. 對被告不利（客觀證明力強）— 10 點
F-03（唯一業務+同群組審核）、F-07（零 STR）、F-08（同網域同群組）、F-10（FA8 客服）、
F-15、F-17（提議）、F-18（函調字號）、F-19（轉傳含被害人字樣文件）、F-20（傳警調訊息，
待蔡居安身分補強）、F-24（假網站，但業界慣例可減輕）

**JDG 第三輪關鍵裁示：**
1. 黑名單三點（F-17/18/19）已形成**完整證據鏈**，是檢方最強火力；辯方唯一出路是切割
   「提議目的」與「實際濫用」，並證明濫用係周正桓所為。
2. F-08 因 myhomepei.com 同網域，**「各家獨立」辯護正式失效**，但「知道是賭博」仍非
   當然成立。
3. F-20 是**槓桿點**——若蔡居安/王藝縈筆錄證實「那個人安＝N 開頭公司聯絡人」，則 F-20
   由「中性偏不利」翻為「重度不利」；反之則維持薄弱。
4. 整體辯護主軸應收斂為：承認「客戶審查不足（行政/過失層次）」，力守「無賭博/洗錢之
   直接故意」，將責任結構導向賴永三（決策、帳戶）與周正桓（系統、黑名單操作、商家接洽）。
