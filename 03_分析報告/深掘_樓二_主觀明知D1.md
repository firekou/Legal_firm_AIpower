# 深掘 · 樓二 — 主觀明知（D1）

> 對全案總表「樓二」的細部拆解。方法：`02_方法論/盧氏三層辯證分析法.md`；
> skill：`dialectic-three-table`。證據瑕疵審查依 `evidence-flaw-review` checklist（文末）。

**目標事實現象：** 賴宗杰主觀上明知（含不確定故意）客戶從事賭博/洗錢犯罪。
**檢方要靠它成立：** 洗錢幫助犯（舊§14／新§19）、幫助賭博（刑§268）的**主觀構成要件**。
**我方導向：** 抽掉唯一承重樁 D1 → 事實現象重構為「認知盲點下的合法遊戲點數代收認知」。

---

## 一、把「明知」這棟樓再拆細：D1 不是一塊，是四段推論鏈

檢方的 D1 表面是一句話，其實是**四段串接的推論**，任一段斷裂，「明知」就到不了：

```
B1a 周正桓自己「認為客戶涉博奕」（他的認知）
 ↓ 傳遞
B1b 周正桓「有告知賴宗杰」（傳遞事實發生）
 ↓ 回應
B1c 賴宗杰答「業界都是這樣」（賴宗杰說過此話）
 ↓ 解讀
B1d 「業界都這樣」＝賴宗杰明知賭博犯罪仍續做（檢方的語意解讀）
```

**盧氏觀察：** 檢方把四段當成一段賣。我方逐段設防，每一段都有獨立破口——而且**起點 B1a
就已經不含「賭博犯罪」的法律內容**，鏈條在源頭就漏水。

## 五、施力 → 抽積木 → 重構 → 導向

**施力點：B1a–B1d（D1 四段鏈）＋ B6 降階。** B2/B3/B4/B5 只證「客觀可疑（可知）」，唯 D1 指
「已知」；逐段擊斷 D1，「明知」失去唯一承重。

**五重削弱（對照四段鏈）：**
1. 武器③（打 B1b）：周正桓**未具結** → §156-II＋釋582，D1 不得單獨定罪。
2. 武器②（打 B1a）：「博奕」≠「賭博」**定義錯置**——源頭認知不含犯罪構成要件之認識（SUB）。
3. 武器②（打 B1a）：周正桓自承「對法律不夠瞭解」→ 傳遞者無正確認知。
4. 武器①（打 B1d）：史育靜證「娛樂城＝遊戲平台」為產業鏈通用語 → 全鏈盲點，非個人明知。
5. 武器①（打 B2/B5）：客觀紅旗（帳戶備註在賴永三、後台串接在周正桓）不在賴宗杰視野 → 無
   容任基礎（刑§13-II 不確定故意的「預見＋容任」缺預見）。

**B6 降階（武器②，SUB）：** 零 STR／CDD 不足屬**行政管制義務違反**，過失不構成洗錢故意犯。

**重構後的事實現象：** 「金恆通內部有法律外行的概括博奕懷疑」——但撐不起「賴宗杰主觀明知
賭博/洗錢犯罪」。事實現象重構為「**認知盲點下的合法遊戲點數代收認知**」。

**導向：** 洗錢幫助 68–78%、幫助賭博 68–78% 無罪（三補強後 75–83%）。這是全案總開關，守住則
樓四之 D4（FA8）、B3 等連帶消解。

## 六、證據瑕疵審查（`evidence-flaw-review` checklist 逐項跑）

### 證據① 周正桓 27-3 偵訊陳述（D1 的唯一來源）★最高價值標的

| checklist 項 | 審查結果 |
|-------------|---------|
| 1 證據定性 | 供述證據；**共同被告**之不利他被告陳述 |
| 2 傳聞法則（§159） | 偵訊中向檢察官所為，屬 §159-1-II 之傳聞例外，須「具結」且無顯不可信；**惟其拒絕具結**，例外門檻不足 |
| 3 補強法則（§156-II＋釋582） | 共同被告陳述＝證人證言，對賴宗杰**須補強**；D1 無錄音/第三人/書面，且**未具結、未經對質詰問** → **不得為唯一有罪證據** |
| — 補強射程 | 現有補強（B2 81.67%、B3 名稱）僅及「客觀環境可疑」，**不及「賴宗杰主觀明知」** → 補強無效 |
| 4 自白任意性 | 非賴宗杰自白，不適用；周正桓自身陳述任意性無爭議 |
| 5 違法取證（§158-4） | 無違法取證爭議 |
| 6 對造單方記載 | 非犯罪集團帳冊，但屬**共同被告單方記憶之對話**，同有「單方、不利他人、自身已認罪有分散動機」風險 |
| 7 數位同一性 | 無數位載體，純言詞回憶 → **可信度更低**（無客觀固定） |
| **結論** | **能力有瑕（未具結）＋證明力不足（補強不及主觀）→ 不能單獨支撐「明知」。此即抽掉 D1 的法理依據。** |

**證據審查總結（樓二）：** 唯一能指向「明知」的證據①（D1）在**證據能力（未具結）**與**證明力
