**檢方要靠它成立：** 洗錢幫助犯、幫助賭博的**主觀構成要件**。
**我方導向：** 抽掉唯一承重樁 D1 → 重構為「認知盲點下的合法代收認知」→ 主觀故意不足。

### 施力 → 重構 → 導向
- **施力點：B1（唯一承重樁）。** B2–B6 只證「客觀可疑（可知）」，唯 B1 指「已知」。
- **D1 五重削弱**（＝抽 B1 這根樁）：
  1. 武器③ 周正桓未具結 → 補強法則（§156-II、釋582）。
  2. 武器② 「博奕」≠「賭博」定義錯置（SUB）。
  3. 武器② 周正桓自承「對法律不夠瞭解」→ 傳遞者無正確認知。
  4. 武器① 史育靜證「娛樂城＝遊戲平台」為全鏈通用語 → 非個人明知。
  5. 武器① 客觀紅旗（帳戶備註、後台串接）不在賴宗杰視野 → 無容任基礎。
- 武器② B6 降階：合規失職＝行政義務違反（過失），非洗錢故意。
- **重構：** 「金恆通內部有法律外行的概括博奕懷疑」——但撐不起「賴宗杰主觀明知賭博/洗錢犯罪」。
  事實現象重構為「認知盲點下的合法遊戲點數代收認知」。B7 由樞紐升為有利。
- **導向：** 洗錢幫助 **68–78%**、幫助賭博 **68–78%** 無罪（三補強後 75–83%）。全案降為過失。

### 施力 → 重構 → 導向（硬點的誠實處理）
- **這是硬點：D1 是賴宗杰自身、有書面的行為，武器②③抽不掉承重樁。**
- 三層處理（誠實鐵律 4：不假裝拆除，只導向止損）：
  1. **構成要件層（武器②/SUB）**：§41 須「意圖不法利益/損害他人」；提議黑名單目的為防詐、
     保護公司 → **欠缺不法意圖**，不該當刑事個資罪，至多行政違規。
  2. **責任分層層（武器①）**：實際蒐集、登載、納入函調字號皆周正桓（H5 自承）；賴宗杰僅
     「提議概念」，**非§41 蒐集/處理/利用之行為人**。
  3. **量刑止損層（MIT）**：縱成罪，個資法可易科罰金（§41）/緩刑（§74，宣告刑壓 2 年以下）；
     設賠償/義務負擔，**與重罪無罪切割**。
- **導向：** 個資法 **42–55% 無罪**；即使成立係非入監輕罪。這是全案唯一結構性殘餘。

# 樓五：通風報信（F-20 傳訊蔡居安）

**目標事實現象：** 賴宗杰向水房金主蔡居安通風報信、協助逃避偵查。
**檢方要靠它成立：** 洗錢共犯、妨礙司法。
**我方導向：** 身分連結存活，但共謀/指揮閉環倒塌 → 業務常規通知，故意不足。

### 施力 → 重構 → 導向
- **施力點：E3（指揮閉環）＋ A3 業務閉環。** 兩樁以武器③（單方記載）＋事實校準抽除。
- **重構：** 蔡居安是「客戶/聯絡窗口」而非「能指揮金恆通的金主」；賴宗杰的傳送＝對等客戶
  關係下通知商家窗口被圈存的業務常規；無證據證明賴宗杰**明知**其水房身分並**意圖**協助掩飾。
- **導向：** 故意要件達不到起訴門檻；F-20 停在「中性偏不利、不成罪」，不牽動重罪。

**兩種達成定義（誠實）：**
- **重罪全免（免入監的實質無罪）：目前 80–87%，三補強後 85–90%。** ✅
- **全罪名零定罪（含個資法）：約 58–68%。** 受樓四拖累，但個資法為可緩刑/易科之輕罪。

**三個可執行補強（把重罪推上 90%）：** ① 聲請傳喚周正桓對質、迫其就 D1 表態具結；② 調閱賴宗杰
完整手機 TG，正面舉證「遊戲點數」認知框架；③ 取得藍新/綠界審查標準書面。

**一句話法庭戰略：** 賴宗杰是被偽裝成遊戲點數商的人頭公司所欺騙的、對等支付服務商之業務招攬
員；他與蔡居安 JIAPAY 水房無人際、無系統、無組織連結（史育靜證實）；帳戶、系統、黑名單操作
皆由他人（賴永三、周正桓）掌控；檢方唯一明知證據 D1 係未具結、法律外行、源於產業鏈被餵養的
「遊戲平台」通用認知，不足證明賴宗杰對賭博/洗錢犯罪有主觀明知。
