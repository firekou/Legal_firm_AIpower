## 這個專案是什麼

這是一個把**單一刑事案件**整理成可被 AI 協作的結構化工作區。核心是一套
**「對抗式多輪會議無罪推論法」**（Adversarial Multi-Round Acquittal Analysis）：
用五個對抗角色（當事人／檢察官／調查局金流分析師／法官／辯護律師）互相辯證、
多輪迭代，把偵查機關認定的每一個「模糊特徵點」逐一拆解，建立一條可在法庭上
站得住的無罪路徑，並誠實標示各罪名的無罪機率與殘餘風險。

截至第十二輪，本案的分析結論：
- **重罪（組織犯罪＋詐欺＋洗錢＋幫助賭博＋銀行法）全部無罪：約 80–87%**，
  執行三項補強後可達 85–90%（＝免於入監的實質無罪）。
- **全部罪名（含個資法黑名單）零定罪：約 58–68%**（個資法為可緩刑/易科罰金之輕罪）。

## 如何使用

1. **理解案件**：先讀 `00_案件總覽/案件事實摘要.md` 與 `涉案人物與角色.md`。
2. **理解方法**：讀 `02_方法論/對抗式多輪會議無罪推論法.md`。
3. **看目前結論**：讀 `03_分析報告/第12輪_無罪路徑總圖.md`（最新收斂結論）。
4. **繼續推演**：在 Claude Code 中，依 `CLAUDE.md` 的指引指派五角色（例如
   `@DEF 針對 F-17 提出反駁`、`@JDG 評估 F-09 的證明力`）開新一輪對抗會議。

## 誠實鐵律

本專案的所有機率估計與策略，遵守以下原則（詳見方法論）：

1. **不為達標而灌水機率** —— 給會在法庭崩潰的數字，是對當事人失職。
2. **分罪名估計** —— 重罪與輕罪分開，避免被殘餘硬點拖累整體判斷。
3. **每根支柱都要站得住卷證** —— 路徑的力量來自可被檢驗，而非樂觀。
4. **硬點正面承擔** —— 當事人自身行為（如黑名單提議）不可假裝消除，只能切割止損。
